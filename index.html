Sobre mim

Sou uma programadora backend júnior com paixão por criar sites e soluções web. Tenho experiência sólida em desenvolvimento utilizando HTML, CSS e JavaScript para criar interfaces modernas e responsivas. Além disso, possuo conhecimentos em linguagem de programação Java, o que me permite criar funcionalidades dinâmicas e escaláveis.
Sou uma profissional dedicada e sempre estou em busca de aprimorar minhas habilidades e aprender novas tecnologias. Sou organizada, colaborativa e capaz de trabalhar bem em equipe, buscando soluções eficientes para os desafios que encontro.
Estou empenhada em desenvolver uma carreira sólida como programadora backend, contribuindo para projetos inovadores e impactantes. Estou aberta a oportunidades que me permitam crescer profissionalmente e aplicar meus conhecimentos em programação web para alcançar resultados excelentes.
Se você está em busca de uma programadora backend júnior com habilidades em HTML, CSS, JavaScript, Java e WordPress, estou pronta para contribuir com sua equipe e tornar seus projetos web um sucesso.
